Refactor `ShapeHook`/`ShapeComponent` and `useShape` to drive all re-renders through a single per-instance update counter instead of one React `useState` per state key. State values live on `this.state` (as they already did for reads) and the counter is bumped only when the component is mounted, so `setState`/`this.s.foo = value`/`shape.set({foo})` after unmount silently updates the instance state without triggering React's "Can't perform a state update on an unmounted component" warning. Removes the need for `if (this.isMounted())` guards around async state writes (e.g. in `.catch` handlers or long-running effects). Mid-render writes still defer through the existing after-paint queue, and `componentDidUpdate` snapshots, `silent` writes, and reference-equality bailouts are unchanged.

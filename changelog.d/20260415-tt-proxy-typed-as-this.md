Type `ShapeHook`/`ShapeComponent`'s `this.tt` proxy as `this` instead of `any`, so subclasses get correctly-typed access to their own methods and fields through `this.tt.someHandler`. Mirrors the existing typing on `this.p` (`this["props"]`) and `this.s` (`this["state"]`) — `npm run typecheck` now catches typos like `this.tt.misspelledHandler` at compile time in addition to the existing runtime throw from the `fetchingObject` proxy.
